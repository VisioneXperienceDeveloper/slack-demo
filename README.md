# Slack Clone — Linear Style

A high-fidelity Slack clone built with Next.js 16, featuring a premium monochrome design inspired by Linear.

## 🚀 Overview

This project is a demonstration of modern web development practices, focusing on **Vertical Slice Architecture**, performance, and aesthetic excellence.

## 📁 Project Structure

For a detailed breakdown of the architecture and directory structure, please refer to the [Architecture Documentation](./docs/architecture.md).

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **State Management**: React Hooks
- **Styling**: Vanilla CSS Modules with custom design tokens
- **Architecture**: Vertical Slice Architecture

## 🏁 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
