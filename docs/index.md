## Project Vision

The goal of this project is to demonstrate a professional-grade web application with:
1. **Premium UI/UX**: Inspired by the minimalist and aesthetic excellence of Linear.
2. **Real-time Persistence**: Leveraging Convex for instant data synchronization.
3. **Scalable Monorepo**: Using Turborepo to manage multiple apps and packages efficiently.
