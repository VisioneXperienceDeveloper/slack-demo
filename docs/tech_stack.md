# Technology Stack

This project is built using modern web technologies focusing on performance, scalability, and type safety.

## Core Technologies
- **Framework**: [Next.js 16.2.4](https://nextjs.org/) (App Router)
- **Library**: [React 19.2.4](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS Modules
- **Package Manager**: NPM

## Architecture
The project follows **Vertical Slice Architecture**, which organizes code around features rather than technical layers (like controllers/models). This approach improves maintainability and makes it easier to understand the context of each feature.

## Development Tools
- **ESLint**: For code quality and linting.
- **TypeScript**: For static type checking.
- **Next.js Dev Server**: Fast refresh and optimized development experience.
